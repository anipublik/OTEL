# examples/

Runnable end-to-end demos that show how otel-starter configs connect to a real application.

| Example | What it demonstrates |
|---------|---------------------|
| [java-payments-service/](java-payments-service/) | Spring Boot + javaagent, custom spans, custom metrics, full Collector → Prometheus → Grafana pipeline |

Each example has its own README with copy-paste commands. Start with the Java payments service if you want to see custom metrics land in Grafana in under ten minutes.
